## 发现的缺陷

| 缺陷ID | 模块 | 关联用例 | 标题 | 严重程度 | 状态 |
|--------|------|---------|------|----------|------|
| BUG-001 | 看板/Kanban | KAN-03 | 无效月份参数返回 500 而非 400 | P1 | 新建 |
| BUG-002 | 变更管理 | CHG-05 | 创建变更时 version_id 为 string 导致 500 错误 | P0 | 新建 |
| BUG-003 | 权限矩阵 | PRM-01 | `/permissions/matrix` 路由未注册，返回 404 | P0 | 新建 |
| BUG-004 | 附件管理 | ATT-01 | 文件上传报 INTERNAL_ERROR | P1 | 新建 |
| BUG-005 | 认证 | AUTH-04 | Token 刷新接口 refresh_token 校验策略不明确 | P2 | 新建 |
